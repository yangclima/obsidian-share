Um campo escalar é um tipo especial de função que define um valor escalar único para cada ponto do espaço, algo como uma [[Funções de mais de uma variável|função de três variáveis]]:
$$
f(x,y,z)
$$
Um bom exemplo seria uma função de 3 variáveis que define a temperatura para qualquer ponto $(x,y,z)$ de um determinado sistema.

Por sua vez, um campo vetorial define um [[MIT - Courses/Classical Mechanic/Vetores|vetor]] para cada ponto do espaço, algo como:
$$
\vec{E} = F_x(x,y,z) \ \hat{i} + F_y(x,y,z) \ \hat{j} + F_z(x,y,z) \ \hat{k}
$$
Um bom exemplo é o campo elétrico gerado por uma partícula carregada, definido como:
$$
\vec{E} =\lim_{q_0 \ \rightarrow \ 0} \dfrac{\vec{F}_e}{q_0} = k_e\dfrac{q}{d^2}\hat{r}
$$
